Installation
============
Install rails.

```
$ bundle install
```


Install grunt, bower, etc.

```
$ cd ngapp/
$ npm install -g gulp bower
$ npm install
$ bower install
```

Run application.

```
# in Rails root
$ rails s

# in ngapp/
$ cd ngapp/
$ gulp
```
